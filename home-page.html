Comicss is a project created by an artist for artists to animate their works in the simplest way possible, it was developed to run in any modern web browser and using only CSS!. For now, it is possible to apply animations and replicate some visual effects for manga comic book style, webcomics and illustrations by creating a simple HTML page and applying the animation and visual effects models created so far.
